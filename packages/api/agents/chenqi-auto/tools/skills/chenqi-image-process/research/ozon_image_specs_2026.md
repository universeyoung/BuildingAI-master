# Ozon 商品图规范与最佳实践 (2026年调研报告)

## TL;DR (3 行)
1. **强制 3:4 比例**：2026年 Ozon 已全面推行 3:4 垂直比例（非传统的 1:1），所有品类（包括电子产品）均需适配此比例以获得最佳展示权重。
2. **主图营销受限**：主图允许信息图（Infographics），但严禁包含价格、折扣百分比、联系方式及夸大词汇（如 "Hit", "Best"）。
3. **俄文原生化**：高转化率依赖于 Roboto/Manrope 字体下的西里尔文卖点标注，以及针对性强的 lifestyle 场景展示。
