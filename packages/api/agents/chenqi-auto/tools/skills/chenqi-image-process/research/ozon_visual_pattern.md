# Ozon 便携蓝牙音箱类目视觉模板分析报告

| 位置 | 常见内容模板 | 出现频率 | 核心特征 |
| :--- | :--- | :--- | :--- |
| **Pos 1** | `主图_带文字` | 90% | 极少使用纯白底。通常带醒目型号名（如 F76）、核心参数（功率、续航）及俄文卖点。 |

## 2. 高销量商品图共有的 5 个视觉特征

1.  **极高饱和度的氛围感**：主图多采用紫色、蓝色或绿色作为背景底色，配合 RGB 灯光效果的渲染，营造电子产品的科技感和派对感。
2.  **大字号卖点直击**：在图片顶部或侧边使用粗体、高对比色的俄文单词（如 **МОЩНАЯ** - 强劲的，**ЧИСТЫЙ ЗВУК** - 纯净音质）。
3.  **配套感展示**：如果是卡拉OK音箱，麦克风和遥控器一定会出现在主图中，且比例较大，突出“全套购齐”的价值感。
4.  **实景代入感强**：非常注重在俄罗斯典型家居环境（暖色木质家具、地毯）中的实拍，拉近与用户的距离。
5.  **核心型号标识**：畅销款通常会在主图显著位置标出型号（如 F71, F76, TBS-02），方便用户二次搜索或对比。

## 3. 与中国 1688 主图的最大 3 个差异

1.  **拒绝纯白底**：1688 上为了适配各种分销渠道，常提供纯白底图；而 Ozon 高销量商品几乎 100% 采用带背景、带文字、带氛围的营销主图。
2.  **卖点密度更高**：1688 主图往往追求简洁，甚至只有产品；Ozon 主图更像是一个微型详情页，第一眼就能看到 3-5 个核心参数。
3.  **场景本土化**：Ozon 上的场景图多为真实的俄罗斯家庭或欧洲风格户外场景，避免了 1688 常见的过于“网红风”或具有明显中国特色的背景。

## 4. 总结建议
针对视觉模板设计，建议首图必须包含：**RGB氛围光 + 粗体型号 + 3个核心数字卖点（瓦数/小时/防水等级）**。后续图片应交替使用**家居场景**和**参数规格表**。
