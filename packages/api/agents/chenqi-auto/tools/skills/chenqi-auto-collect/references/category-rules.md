## 匹配规则

1. 优先匹配更具体的类目（如"收纳篮三层"优先于"收纳篮单层"）。
2. 若标题同时含多个类目关键词，取最具体的类目。
3. 无匹配时，使用通用默认值：重量 200g，尺寸 20×15×5cm。
4. 回填后在描述简介末尾追加：`[AI估算]重量/尺寸为系统推算，建议人工核实`。
